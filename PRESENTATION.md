## Stai ancora usando l'Ads Manager?

Ogni giorno apri l'Ads Manager, navighi tra tab, applichi filtri, esporti CSV, copi numeri su Google Sheets, formatti report. Per ogni cliente. Per ogni account.

Non è strategia. È data entry.

E più clienti prendi, più tempo bruci in operazioni che non generano valore.

## Questo strumento elimina tutto quel lavoro

Un server MCP che collega Claude Code alle API di Meta. Tu scrivi in italiano cosa vuoi fare. Claude lo esegue sulle API e ti restituisce il risultato.

Nessuna interfaccia. Nessun click. Nessun export manuale.

36 strumenti che coprono tutto l'Ads Manager: campagne, ad set, inserzioni, analytics, audience, creative, report.

## Report pronti in 30 secondi

Il report è la tassa nascosta di ogni media buyer. Ore spese a tirare fuori dati, formattare, commentare. Ogni mese. Per ogni cliente.

Con un prompt:

```
Report mensile per il cliente.
Spesa totale, lead, CPL, breakdown per campagna.
Confronto con il mese precedente. Evidenzia cosa ha funzionato e cosa no.
```

Claude genera il report completo: tabelle, delta percentuali, commenti interpretativi. Pronto da inviare.

```
Report settimanale. Solo campagne attive.
Per ognuna: spesa, risultati, costo per risultato, trend vs settimana scorsa.
Segnalami quelle che stanno peggiorando.
```

```
Esportami gli ultimi 90 giorni in CSV con breakdown per età, genere e piattaforma.
```

```
Confronta Gennaio vs Febbraio su tutti gli account attivi.
Tabella riassuntiva con delta percentuali.
```

Il report non è più un costo. È un comando.

## Creazione ed esecuzione

```
Crea una campagna lead gen.
Target: donne 25-45, Italia, interesse marketing digitale.
Budget: 50 euro/giorno.
```

```
Metti in pausa [campagna A]. Raddoppia il budget su [campagna B].
```

```
Crea una lookalike 2% Italia dalla custom audience clienti alto valore.
```

```
Quante persone raggiungo con target uomini 30-50, Germania, interesse SaaS?
```

Ogni operazione di creazione o modifica segue un protocollo: Claude ti fa le domande necessarie, ti presenta il piano in tabella, aspetta la tua conferma, e solo dopo esegue.

Le campagne nascono sempre in pausa. Nessuna attivazione senza il tuo ok.

## Come funziona sotto il cofano

**Currency nativa** — Legge la valuta dell'account dall'API. Account in euro mostra euro. Account in dollari mostra dollari.

**Multi-account** — Tutti gli account collegati al token sono accessibili. Switch istantaneo tra un cliente e l'altro.

**Sicurezza** — Ogni campagna nasce in pausa. Ogni modifica viene pianificata, mostrata in tabella e confermata prima dell'esecuzione.

**Auto-documentazione** — Il file `CLAUDE.md` insegna a Claude le regole dell'API e i workflow corretti. Lo legge automaticamente. Zero configurazione.
